# VideoVault Frontend (Mocked)

This is a premium React-based frontend demonstration for the VideoVault platform.

## Tech Stack
- **Framework**: React 18 + Vite
- **Styling**: Tailwind CSS v4 + Glassmorphism
- **Icons**: Lucide React
- **Animations**: Framer Motion
- **State Management**: Zustand

## Features Demonstrated
- **Premium Dashboard**: Dark cinematic theme with glassmorphism effects.
- **Video Fetching**: Mocked logic for extracting and analyzing video URLs.
- **Platform Detection**: Automatically identifies Douyin, TikTok, XHS, and Bilibili.
- **Download History**: Interactive list of previous downloads with status indicators.
- **Quota Management**: Visual representation of remaining download quota.
- **Mock Progress**: Real-time progress simulation for new downloads.

## Getting Started

1. **Install Dependencies**:
   ```bash
   cd frontend
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

3. **Build for Production**:
   ```bash
   npm run build
   ```

## Design Principles
- **Wow Factor**: Vibrant primary colors, deep backgrounds, and smooth transitions.
- **Clarity**: High-contrast typography and intuitive layouts.
- **Feedback**: Immediate visual response for all user actions (fetching, downloading).
