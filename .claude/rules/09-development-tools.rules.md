# Development Tools (Recommended)

## 9.1 Core Development Environment
- **IDE**: Visual Studio Code (with extensions: C# extension, Docker extension, ESLint, Prettier)
- **Version Control**: Git (with SSH keys for GitHub access)
- **Package Manager**: npm (for frontend), dotnet (for backend)
- **Database GUI**: DBeaver (PostgreSQL connections)
- **API Testing**: Postman / Insomnia
- **Container Management**: Docker Desktop (Windows/macOS) / Podman (Linux)
- **CLI Tools**: Windows Terminal / iTerm2 / Windows PowerShell 7+

## 9.2 Productivity Tools
- **Browser**: Google Chrome (with DevTools) or Firefox Developer Edition
- **REST Client**: Thunder Client (VS Code extension) for quick API testing
- **Documentation**: MkDocs or Docusaurus for internal docs
- **Code Quality**: 
  - Linting: ESLint (frontend), StyleCop (C#)
  - Type Checking: TypeScript (frontend), nullable reference types (C#)
  - Formatting: Prettier, dotnet-format

## 9.3 CI/CD Tools
- **GitHub CLI**: For repository management
- **Docker Registry**: GitHub Container Registry (GHCR) or Docker Hub
- **Secret Management**: GitHub Secrets, HashiCorp Vault, or AWS Secrets Manager

## 9.4 Monitoring & Debugging
- **Log Viewer**: Loki + Grafana or ELK stack
- **Network Inspection**: Wireshark or tcpdump
- **Performance Profiling**: 
  - .NET: dotTrace, dotMemory
  - Node.js: Clinic.js, Node Clinic
- **Browser Testing**: Selenium, Playwright

## 9.5 Collaboration Tools
- **Issue Tracking**: GitHub Issues / GitLab Issues
- **Project Management**: Jira, Linear, or GitHub Projects
- **Code Review**: GitHub Pull Requests, GitLab Merge Requests
- **ChatOps**: Slack with GitHub integration
